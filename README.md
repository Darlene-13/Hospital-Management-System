# Hospital-Management-System
Hospital Management System in Python and Java maybe later to master OOPs and Methods

### Project Overview
This project aims to develop a comprehensive a Hospital Management System using Python and Java.
The primary goal is to enhance my understanding of Object-Oriented Programming (OOP) concepts and methods in both programming languages
While creating a functional application, I intend to solidfy my coding skills and improve my problem-solving abilities.


### Project Setup
The project will be organized into separate directories for Python and Java implementations to avoid any compatibility issues.


### Dependencies
1. Python Dependencies:
    - Flask
    - SQLAlchemy
    - aiosqlite
2. Java Dependencies:
    - Spring Boot
    - Hibernate
    - MySQL Connector
3. Vanilla Java for core OOP concepts without any frameworks.
